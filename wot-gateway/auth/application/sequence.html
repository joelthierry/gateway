Sequenzen

Hier können Sequenzen für den Motor zusammengestellt werden. Aktion auswählen, Werte eintragen und dann hinzufügen. Wenn alle gewünschten Aktionen hinzugefügt sind, kann die Sequenz an den Motor gesendet werden.
Wenn die Sequenz auf den Motor geladen wurde, kann die Sequenz ausgeführt werden.
